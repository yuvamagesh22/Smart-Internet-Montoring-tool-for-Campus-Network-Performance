Smart-Internet-Montoring-tool-for-Campus-Network-Performance:
A Smart Internet Monitoring Tool is an intelligent network management system designed to monitor the health, traffic, and performance of campus network infrastructure. The system continuously collects network data and analyzes it to detect performance issues, congestion, or security threats.

Features Of The Project:
1. Internet Speed Testing
The system measures download speed, upload speed, and ping to analyze the current network performance.

2. Public IP Detection
It automatically detects and displays the user’s public IP address.

3. Data Storage Using Database
All speed test results are stored in a SQLite database, allowing users to track network performance over time.

4. History Tracking
Users can view previous speed test results, which helps in identifying patterns or network issues.

6. Optimization Suggestions
The program provides recommendations to improve network performance, such as restarting the router or using Ethernet.

7. Notification System
After completing the speed test, the system sends a desktop notification with the results.

Problems Solved:

1. Difficulty in Identifying Network Issues
Students often complain about slow internet without actual data. This project provides accurate speed measurements.

2. Lack of Network Monitoring
Most campuses do not have simple tools for students to monitor internet performance. This system allows easy network monitoring.

3. No Historical Network Data
Without stored results, it is difficult to analyze network performance. The database helps track internet performance over time.

4. Delayed Problem Identification
By analyzing speed results and ping values, users can quickly identify connectivity problems.

5. Poor Awareness of Network Optimization
The program provides simple suggestions to help users improve their connection quality.
